Publish

Clicking Publish to GitHub fetches your live index.html, swaps the STORE data with your edits, and commits it back. GitHub Pages rebuilds in ~30 seconds.

How it works

1. Fill in Token, username, repo name, file path, and branch above.

2. Click Test Connection → should show "Connected ✓".

3. Click Load from GitHub to import your current live store data.

4. Edit products, hero text, categories in the other tabs.

5. Click Publish to GitHub → site goes live in ~30 seconds.
