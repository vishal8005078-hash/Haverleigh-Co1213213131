GitHub & Publish
Connect your GitHub repo. Publish writes directly to your live site.
How it works
1
Go to github.com/settings/tokens → Classic token → tick repo → Generate → copy the token.
2
Paste it below with your repo details. Click Test Connection to verify.
3
Click Load from GitHub to pull your current live store data into all tabs.
4
Edit anything in the other tabs.
5
Click Publish to GitHub. Your live site updates in ~30 seconds via GitHub Pages.
