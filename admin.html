How to Publish Your Website
3 simple steps — takes about 2 minutes the first time, 30 seconds after that.
First Time Setup (do once)
1
Go to github.com and create a free account if you don't have one.
2
Create a new repository. Name it anything — like haverleigh-site. Set it to Public.
3
Go to the repo's Settings → Pages. Under "Branch" select main and click Save. Your site URL will appear here (like yourname.github.io/haverleigh-site).
Every Time You Update
1
Edit your content here in the builder. Then click Download Website — you'll get a file called index.html.
2
Go to your GitHub repository. Click Add file → Upload files.
3
Drag and drop the downloaded index.html into GitHub. If an old one exists, it gets replaced automatically. Click Commit changes.
4
Wait about 30–60 seconds, then refresh your site URL. Your changes are live! 🎉
Important Note

All images you upload from your device are embedded directly into the HTML file — no external hosting needed. The downloaded file is completely self-contained. Just upload it to GitHub and it works.
